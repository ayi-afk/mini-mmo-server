# mini-mmo-server
The server for a very simple open world (not massively) multiplayer online game

## Requirements
* Protobuf compiler
* Python 3.7

A virtualenv is also highly recommended.

## Usage
To install:

```
$ make
```

To run:

```
$ game-server
```

Currently the `-v` and `-p <port>` arguments are available - verbose and port number.
